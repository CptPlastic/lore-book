# FAQ

## Is lore cloud-based?

No. Lore stores memory locally in your repository.

## Does lore require API keys?

No.

## Which AI tools are supported?

Lore exports a central chronicle and lean adapter files for several popular assistants.

## How do I migrate docs from another repo?

Copy your markdown files into `docs/`, then add them to `nav` in `mkdocs.yml`.
