Install

Get Lore running locally in a few commands.

✦ Requirements

Python 3.10+.

⚜ Install From PyPI

pip install lore-book

✧ Local Development Install

pip install -e .

⌘ Optional Semantic Search

pip install "lore-book[semantic]" lore setup semantic
