Lore

Local AI memory for your codebase.
Chronicle decisions, context, and lessons your tools can actually read.

pip install lore-book
then
lore onboard

AI tools are stateless. They don't remember why you chose Postgres, what rules can't be bypassed, or the tradeoffs that already shaped your codebase. Lore gives them persistent memory — stored as plain YAML next to your code, no cloud, no API keys.

Record knowledge

lore add decisions "Use Postgres for row-level locking" lore add facts "Auth service is the sole JWT issuer"

Search and export

lore search "why postgres" lore export # → CHRONICLE.md

CHRONICLE.md is automatically wired into Copilot, Claude, Cursor, and AGENTS instruction files on export.
